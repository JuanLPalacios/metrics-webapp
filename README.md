# Metrics web app

mobile web application to check a list of narrativa API COVID-19 metrics.

### Setup


to set it up yo have to first run 

```
npm install
```

after that you can run the local development server with:
```
npm start
```

## Built With

- React
- Redux
- JavaScript
- CSS
